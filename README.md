# Knick Knack Labs Stories

Custom static site for KKL stories, essays, and prototype writeups.

Planned public URL: https://stories.knacklabs.co/

## First pass

- Astro static site
- Markdown content collection for stories
- Square/grid visual language borrowed from the first KKL-ish PoCs
- Buttondown subscribe placeholder; Buttondown will be the mailing list, not the blog engine

## Run locally

```bash
bun install
bun run dev
```

Open http://localhost:4321/.

## Checks

```bash
bun run check
```

## Notes

This is a first-pass scaffold. Next pass should review the fold/new-codebase guidance before hardening project conventions, deploy, CI, content workflow, and domain setup.
